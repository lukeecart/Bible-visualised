The Visual Bible

Feel free to use some of these resources to learn more about the bible.

During the last year or so I have been wanting to create a visual guide to the bible. I have created this website to showcase some projects that have been built to visualise family tree data from the bible, and a timeline of the events of the bible. The data has been collected from multiple sources.
